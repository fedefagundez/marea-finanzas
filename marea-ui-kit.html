Marea
Finanzas del hogar, en calma.

Un lenguaje visual turquesa, cálido y con carácter para llevar el control de gastos, ingresos y metas de ahorro familiares.

Fundamentos

Paleta de colores

El turquesa es el protagonista: transmite liquidez, calma y control. El índigo aporta contraste y jerarquía. Los colores semánticos comunican el estado de cada movimiento de un vistazo.

Primary — TurquesaMarca, acciones principales, foco
Secondary — ÍndigoContraste, elementos secundarios
Neutral — Gris azuladoFondos, bordes, texto
SuccessIngresos
DangerGastos
WarningPendientes
Fundamentos

Tipografía

Sora para títulos y montos destacados — geométrica, con presencia. Inter para texto de cuerpo e interfaz — máxima legibilidad. JetBrains Mono para cifras en tablas — alineación perfecta de dígitos.
